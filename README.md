# Instream

InfluxDB driver for Elixir


## Warning

__This module is highly experimental at the moment and may behave or change unexpectedly.__


## Setup

Add Instream as a dependency to your `mix.exs` file:

```elixir
defp deps do
  [{ :instream, "~> 0.1" }]
end
```
